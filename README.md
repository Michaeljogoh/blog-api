# blog-API 
Blog application is a rest-api 

# dependencies
- express.js
- mongoose
- dotenv

## Features

- Utilizes the Express web framework.
- Uses MongoDB on the backend.
- Easy Heroku integration.
- Create, edit, and delete your posts.
